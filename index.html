Simple 4-Step Setup

Follow these steps to get QueryCraft running on your machine

1

Clone & Install

Get the QueryCraft repository and install all dependencies

git clone https://github.com/mannat244/QueryCraft.git
cd QueryCraft
npm install
2

Get API Keys

Obtain your free API keys from Google Gemini and Pinecone

Google Gemini API

Free AI model for natural language processing

Get API Key

Pinecone Vector DB

Vector database for schema storage and retrieval

Get API Key
3

Configure Environment

Create your environment file with the API keys

# .env.local
SESSION_PASSWORD=your-secure-password-here
GEMINI_API_KEY=your-gemini-api-key-here
PINECONE_APIKEY=your-pinecone-api-key-here
INDEX_NAME=schemaindex

# Optional: Database SSL Certificate
DB_SSL_CA="-----BEGIN CERTIFICATE-----
your-ssl-certificate-content-here
-----END CERTIFICATE-----"
